0. Intro

This program uses an Arduino + the ESP8266 board as a WiFi communication device. The Arduino uses the HttpServer component in order to provide an HTML user interface. The user interface controls 2 relays. Additionally, these 2 relays are also commanded by 2 push buttons. When pushing them, the ON/OFF status is updated in the HTML interface.

1. Hardware Used

  • Arduino Micro;
  • ESP8266 Board (ESP-201 variant);
  • Relay Board;
  • 2 x Push Buttons (brick);
  • 2 x 3.3KOhm Resistors;
  • Breadboard;
  • Extension Headers (to attach the ESP8266 board to the breadboard).

2. Firmware of the ESP8266

The ESP8266 is a versatile SoC (System on Chip). It can have different variants of firmware, and it can even be programmed from the Arduino IDE, just like an Arduino. In our case we need the AT Commands firmware.
  • This article contains detailed information about how to flash the AT Commands firmware to your ESP8266.
  • And this article explains how to check to see if you have the latest AT Commands firmware installed.
