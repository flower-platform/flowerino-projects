Event type: ValueChangedEvent

Available "invocations":
"IF value == HIGH => Invoke": converts from ValueChangedEvent to () and invokes the operation
Invokes the operation only when the value from the event is HIGH.
"IF value == LOW => Invoke": converts from ValueChangedEvent to () and invokes the operation
Invokes the operation only when the value from the event is LOW.
"Always => Invoke (relaying "value")": converts from ValueChangedEvent to (value) and invokes the operation
Invokes passing the value from the event. I.e. the value is relayed without any processing.
