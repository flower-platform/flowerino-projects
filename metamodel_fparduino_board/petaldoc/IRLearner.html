IRLearner

Jump to: Properties Operations Events

Used for capturing IR commands from theoretically any infrared remote control available on the market and storing them on the SD card, under the /ir-commands folder.
Makes use of an IR receiver, which must be connected to one of the pins within 0..7 range.
An associated IRLearnButton on the UI is needed in order to use this petal.

Properties
name:String
Mandatory. E.g.: myNewObject.
The name of the object, which needs to be a valid C++ identifier.
pin:Number
Mandatory. E.g. 3 or A3.
The pin of the input. When choosing pins, please see the remark above (about occupied pins).
Operations
capture(name)
Captures IR command and saves it to SD card, under the /ir-commands folder. The name argument is used as the file name.
Events
None.
