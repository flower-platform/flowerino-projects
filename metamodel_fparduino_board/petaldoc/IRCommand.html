IRCommand

Jump to: Properties Operations Events

Used to send an IR command.
The IR LED must be connected to pin 3 on most boards, or to pin 9 on the Mega. For specific chips, please see the IRremote library.
The name property is also used as the name of the file on the SD card where the command is stored.
Commands are looked up in the /ir-commands folder on the SD card.

Properties
name:String
Mandatory. E.g.: myNewObject.
The name of the object, which needs to be a valid C++ identifier.
Operations
send()
Transmits the IR command sored in the file corresponding to the name property of the petal, under the /ir-commands folder.
Events
None.
